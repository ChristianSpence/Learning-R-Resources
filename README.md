# Learning R Resources
My go-to list of resources for learning, revising and improving your R knowledge and workflows

## Books

### General introductions to R and Data Science/Analytics in R
* [R for Data Science](https://r4ds.had.co.nz/)
* [Advanced R](https://adv-r.hadley.nz/)
* [R Cookbook](https://rc2e.com/)
* [Hands-on Programming with R](https://rstudio-education.github.io/hopr/)

### Packages
* [R Packages](https://r-pkgs.org/)

### Data Visualisation
* [ggplot2: Elegant Graphics for Data Analysis](https://ggplot2-book.org/)
* [R Graphics Cookbook](https://r-graphics.org/)

### Spatial and GIS
* [Geocomputation with R](https://geocompr.robinlovelace.net/)
* [R as GIS for Economists](https://tmieno2.github.io/R-as-GIS-for-Economists/)

### Git and GitHub
* [Happy Git and GitHub for the useR](https://happygitwithr.com/)

### Shiny
* [Mastering Shiny](https://mastering-shiny.org/)
* [Engineering Production-Grade Shiny Apps](https://engineering-shiny.org/)

### Time series
* [Forecasting: Principles and Practice](https://otexts.com/fpp3/)

### Modelling
* [Statistical Inference via Data Science](https://moderndive.com/)

### R Markdown
* [R Markdown: The Definitive Guide](https://bookdown.org/yihui/rmarkdown/)
